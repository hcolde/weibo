## 关于这个程序

### 说明

  利用requests库爬取某个博主的所有微博(包括发微博的时间)，保存到xls表格里

  爬取微博PC端很有难度，所以这里爬取的是手机端

  这个程序只供学习，不然，小拳拳捶你胸口！

### 一些不足

  这个程序是单进程爬取，而一些大V的微博有好几千条甚至上万条，所以爬完大V的微博耗时很久很久...

### 以后的改进

  因为考虑到python没有真正的多线程这一说，改善耗时太久这一问题，我打算利用多进程爬取，因为在未来的时间我将会继续学习多进程的知识
